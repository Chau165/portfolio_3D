# Portfolio 3D

React + Vite portfolio project prepared for local development, GitHub source control, and Vercel deployment.

## Tech Stack

- React
- Vite
- TypeScript
- Three.js
- @react-three/fiber
- @react-three/drei
- GSAP
- Framer Motion
- Lenis

## Setup

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Vercel Deploy Notes

- Framework preset: `Vite`
- Build command: `npm run build`
- Output directory: `dist`
- Keep `package.json`, `package-lock.json`, `index.html`, `src/`, `public/`, and `public/models/` in the repository.
- Do not remove required 3D assets from `public/models/`.
